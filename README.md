# robochat

Speak to person of your choice!

## run

Requires node, express, socketio. Run `node server.js` then open http://localhost:8080/client.html.

Requirements for python:
- python3
- pip3 (package manager)
- markovify 1.1.1
- simplejson 3.16.0
